## PropStream (propstream.com)

### UX & UI Patterns
1. **Dashboard** → Location search (zip/city/state)
2. **100+ filter accordions** (equity %, liens, foreclosures) with checkboxes/sliders
3. **Live counts** update via take=0 API, right sidebar shows active filters
4. **Preview sample records** → "Build List" → credits cost → async queue (>50k)
5. **Orders tab** tracks completion

**Strengths:** Instant feedback builds trust
**Weaknesses:** Filter overload confuses beginners

### Feature Gaps
| Feature | Supported | Notes |
|---------|-----------|-------|
| Suppression/deduplication | Yes | Past lists |
| Async large downloads | Yes | >50k queued |
| Cost estimate | Yes | Credits preview |
| User management | Partial | Team plans |
| White-label/API | Partial | API only |

### User Complaints
1. **Export limits** (10k/day) — G2, Reddit
2. **Stale data** (old liens) — Capterra
3. **Slow support** — r/realestateinvesting

## BatchLeads (batchleads.io)

### UX & UI Patterns
1. **Search bar** (address/zip)
2. **Quick Lists** (prebuilts) or Basic/Advanced tabs
3. **AND/OR toggles**, category live counts
4. **Manual property selection** → credits preview → Generate List
5. **Instant CSV** (small) or async (large)

**Strengths:** Quick Lists speed up common searches
**Weaknesses:** No maps, credit gates block previews

### Feature Gaps
| Feature | Supported | Notes |
|---------|-----------|-------|
| Suppression | Partial | CSV upload only |
| Async downloads | Partial | Scale limits apply |
| Cost estimate | Yes | Credits shown |
| User management | Partial | Basic only |
| White-label/API | No | None |

### User Complaints
1. **Pricing surprises** — Trustpilot
2. **Mobile filters unusable** — G2
3. **Duplicates persist** — r/wholesaling

## DealMachine (dealmachine.com)

### UX & UI Patterns
1. **Map overlay** → draw polygons/radius
2. **700+ filter dropdowns/chips**
3. **Live pin bubbles**, table distance sort
4. **Leads tab** refine → one-click CSV + skip trace
5. **No cost preview before export**

**Strengths:** Visual discovery via map
**Weaknesses:** Desktop lag, app crashes

### Feature Gaps
| Feature | Supported | Notes |
|---------|-----------|-------|
| Suppression | Partial | Phone-based only |
| Async downloads | No | Plan limits |
| Cost estimate | No | Skip trace upsells instead |
| User management | Partial | App invites only |
| White-label/API | No | Mobile-first product |

### User Complaints
1. **Map lag on large areas** — Capterra
2. **Forced skip trace upsells** — Reddit
3. **iOS crashes** — Trustpilot

## Strategic Insights

### Biggest UX Pattern to Adopt
**PropStream live counts + filter chips** — Instant feedback prevents users from building bad lists and reduces failed downloads.

### Biggest UX Pattern to Avoid
**BatchLeads credit gates** — Blocking previews behind credits kills momentum. Use transparent async flow instead.

### Clearest Gap to Exploit
**True white-label with API token dataset control** — No competitor offers full reseller branding with per-client data access control.

### Biggest Competitive Risk
**PropStream filter depth** (100+ criteria) — Hard to match without schema drift. Requires a clear schema update process with PropExport.
